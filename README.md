dfb-wireshark-dissector
=======================

Wireshark dissector for Voodoo protocol, network layer of DirectFB (http://directfb.org/index.php?path=Platform/Voodoo).

Current features:
- Both packet & raw modes are supported ;
- FLZ decompression ;
- Instance ID resolution.
